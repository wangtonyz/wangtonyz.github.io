# Professional About Me Portfolio

A modern, interactive portfolio website

## Features

- **Modern Design**: Dark theme with vibrant accents and sleek animations
- **Interactive Elements**: Animated skill bars, timeline animations, and hover effects
- **Fully Responsive**: Optimized for all device sizes
- **Performance Focused**: Lightweight and fast loading
- **Geek Style**: Terminal-inspired UI elements and code-themed design

## Technologies Used

- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts
- Qwen Code AI for content generation
- Roo Code for code snippets
